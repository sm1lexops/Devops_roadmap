## Go

### Cheat Sheet

* Run your program:

```
go run <file path>
```

* Compile your app package:

```
go build <package path>
```

* Install and run from a remote location:

```
go install github.com/bregman-arie/myApp
bin/myApp
```

* var definition

```
var i int = 42
OR
i := 42
```

* Define several variables

```
var (
     name string = "hello"
     num int = 42
```
